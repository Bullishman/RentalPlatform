## 렌탈 플랫폼 프로젝트
- 유저 간 빌려주고 싶은 혹은 빌리고 싶은 물건들을 게시글을 바탕으로 대여를 할수있는 기능을 가진 애플리케이션입니다.

---
## 기술스택
- OS: Windows
- Frontend: React
- Backend: Java, SpringBoot, JPA, SpringCloud(Gateway, Eureka, Config, Kafka)
- DB: MariaDB
- VCS: Git
- IDE: IntelliJ, VSC
